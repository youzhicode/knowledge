Linux实用指令

  • 帮助指令
    • man 指令 eg: man ls
    • help 基本语法： help xxx
  • 指令运行级别
    • 0：关机级别
    • 1：单用户（找回丢失密码）
      • 面试题，如何找回root用户密码
    • 2：多用户无网络服务
    • 3：多用户有网络服务
    • 4：保留
    • 5：图形界面
    • 6：重启
    • 系统运行级别的配置文件 /etc/inittab
    • 如何指定系统运行级别: init [0-6]
  • 文件目录类
    • pwd 显示当前工作路径 eg: pwd
    • ls 查看目录或者文件的内容信息
      • 基本语法: ls xxx
      • opt:： -a 查看所有文件，包括隐藏文件
      • opt:： -l 以长列表的形式查看文件或者目录的详细信息
    • cd 切换工作目录
      • 常用用法： cd 或者 cd ~ 回到自己的家目录
      • cd xxx 进入到指定的目录
      • cd .. 回到上级目录
      • 支持相对路径与 绝对路径
